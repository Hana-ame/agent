2026-03-12

## 期望用法
命令格式：`py utils.py mv <源路径> <目标路径>`

输出格式：
- 成功时：`成功：已将 '<源路径>' 移动到 '<目标路径>'`
- 失败时：
  ```
  === <源路径> ===
  错误：具体错误信息
  === end of <源路径> ===
  ```

## 实际用法（修改后）
与期望一致。

## 测试结果

### 1. 重命名文件
命令：
```
py utils.py write mv_source.txt "This is a test file."
py utils.py mv mv_source.txt mv_renamed.txt
```
输出：
```
成功：已将 'mv_source.txt' 移动到 'mv_renamed.txt'
```
后续查看文件内容正常，✅ 符合期望。

### 2. 移动到已存在的目录
命令：
```
py utils.py write mv_dir/test.txt "dummy"  # 创建目录和文件
py utils.py mv mv_renamed.txt mv_dir/
```
输出：
```
成功：已将 'mv_renamed.txt' 移动到 'mv_dir\mv_renamed.txt'
```
目录列表显示文件已移入，✅ 符合期望。

### 3. 源路径不存在
命令：`py utils.py mv non_existent.txt destination.txt`
输出：
```
=== non_existent.txt ===
错误：源路径不存在
=== end of non_existent.txt ===
```
✅ 错误块格式正确。

### 4. 目标父目录不存在
命令：`py utils.py mv mv_dir/test.txt non/existent/dir/file.txt`
输出：
```
=== mv_dir/test.txt ===
错误：目标路径的父目录不存在 'non/existent/dir'
=== end of mv_dir/test.txt ===
```
✅ 错误块格式正确。

### 5. 移动目录
命令：`py utils.py mv mv_dir/ mv_moved_dir/`
输出：
```
成功：已将 'mv_dir/' 移动到 'mv_moved_dir'
```
目录列表显示目录已移动，✅ 符合期望。
