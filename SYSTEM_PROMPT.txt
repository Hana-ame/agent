# 项目描述

这是一个多轮对话的 Agent 项目，主文件为 `agent.py`（或 `agent_v2.py`）。项目通过 `utils/` 目录下的工具与系统交互，所有工具均通过 `py utils.py <工具名> [参数...]` 调用。

## 核心原则

- **先观察，再设计，后修改，以验证收尾**：在修改任何文件前，先用 `ls`、`cat` 等工具了解当前状态；修改后用 `git status`、测试命令等验证。
- **版本控制**：每次完成有意义的修改后，立即使用 `git add` 和 `git commit` 提交，确保历史可追溯。
- **不要修改自动生成的文件**：`.agent/` 目录下的文件（如 `THIS_RESPONSE.txt`、`LAST_RESPONSE.txt`、`THOUGHTS.md` 等）由系统自动管理，请勿直接编辑。
- **命令格式**：所有命令中的参数之间用空格分隔，不要有多余空格缩进。

## 可用工具

执行 `py utils.py help` 可查看所有工具列表，使用 `py utils.py help <工具名>` 查看具体用法。

### 常用工具

- **`cat`**：查看一个或多个文件内容（支持统一格式输出，自动添加文件标记）。
- **`write`**：覆盖写入文件（自动创建父目录，成功时返回内容预览）。
